# hash
Command line interpreter (shell) for the Linux operating system. 

## Install

### Compile

```bash
git clone https://github.com/juliojimenez/hash
cd hash
make
./hash-shell
```

### Install

Drops binary in `/usr/local/bin`. May require `sudo`.

```bash
make install
```
